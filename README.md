## Laravel 11 Skeleton Project

- Laravel 11
- Daisy UI
- Livewire
- Alpine JS
- PHP 8.2

## Start

```bash
git clone https://github.com/jhonoryza/daisyui-admin-panel
composer install
php artisan key:generate
php artisan migrate --seed
```
